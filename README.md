# Scoop Bucket

A [Scoop](https://scoop.sh) bucket for apps by [Fanis Hatzidakis](https://github.com/fanis).

## Usage

```powershell
scoop bucket add fanis https://github.com/fanis/scoop-apps
scoop install claude-code-switcher
```

## Apps

| App | Description |
|---|---|
| [claude-code-switcher](https://github.com/fanis/claude-code-switcher) | Fast native Windows utility for switching between Claude Code projects |
